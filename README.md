## Install

`go install github.com/fmo/pomodoro@latest`

## How to run

`pomodoro start --duration 15m`

## Get today's stats

`pomodoro stats --period today`

## Help

`pomodoro help`

## Disable backups

`pomodoro config --backups false`  

## Change file names

`pomodoro config --pomodoro-file pomodoro.csv`  
`pomodoro config --backup-file bup.csv`
